# ⚡️ Imóveis API + PHP + Laravel + Swagger API + Auth Sanctum + Docker + Redis + MySQL 🔥

This is a project to Imóveis with API REST

## Technologies and Tools

✔️ PHP 8.2\
✔️ Laravel 10\
✔️ Swagger API\
✔️ Redis\
✔️ Docker (Laradock or Sail)\
✔️ MySQL

## Project Setup

Setup With Docker Development:

```sh
docker-compose up -d

```

Setup with Artisan

```sh
composer install

php artisan start
```
